# serve

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

for some endoints, you will need an authenticated RPC url

```bash
export RPC_URL=blah
```

This project was created using `bun init` in bun v1.0.4. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
